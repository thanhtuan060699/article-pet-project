# Article Project - Demo

## Backend

### Technical

* Nodejs
* ExpressJS
* Mongodb

### How to run

* Clone source code
* Install packages: `npm install`
* Run BE server: `npm run start`
* Go to: [localhost:8442](http://localhost:8442)
## Frontend

### Technical

* Reactjs
* Nextjs
* Formik
* Chartjs
* Draft-js

### How to run

* Clone source code
* Install packages: `npm install`
* Run FE: `npm run dev`
* Go to: [localhost:3000](http://localhost:3000)
